# StylerX CLI

Live Theme Engine for OBS Studio - Command Line Interface

## Usage

```bash
stylerx install              Install StylerX plugin
stylerx uninstall            Remove StylerX plugin
stylerx update               Check for updates
stylerx version              Show version info
stylerx doctor               Run system diagnostics
```

## Requirements

- Python 3.8+
- OBS Studio 30+

## Installation

### Windows
```
python stylerx.py install --build-dir ../build
```

### Linux / macOS
```
python3 stylerx.py install --build-dir ../build
```
