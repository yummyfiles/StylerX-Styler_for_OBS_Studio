**Style your OBS. Live.**

</div>

Create, customize, and preview OBS Studio themes in real time. StylerX layers a user-defined Qt stylesheet on top of OBS's base appearance вҖ” tweak colors, edit raw QSS, and inspect widgets from a docked panel inside OBS.

---

## вңЁ Features

<table>
<tr>
<td width="50%">

### рҹҺЁ Live Color Picker
Adjust 20+ UI colors вҖ” backgrounds, buttons, accents, text, tabs, scrollbars вҖ” with hex inputs, RGB/A sliders, or HSV wheels. Changes preview instantly.

### рҹ“қ QSS Editor
Full-featured code editor with syntax highlighting, search/replace, line numbers, and <kbd>Ctrl+S</kbd> to apply.

### рҹ”Қ Widget Inspector
Hover to highlight any OBS widget, click to inspect its stylesheet, class name, and object name.

</td>
<td width="50%">

### рҹ’ҫ Theme Library
Save, duplicate, rename, favorite, export, and import themes as JSON. Stored in `%APPDATA%/StylerX/themes/`.

### вҡЎ Debounced Apply
Color updates are debounced (configurable) so bulk edits don't cause flicker.

### рҹ–ҘпёҸ CLI Companion
Command-line tool for installing, diagnosing, and managing the plugin.

</td>
</tr>
</table>

## рҹ”§ Build

```powershell
# 1. Clone
git clone https://github.com/yummyfiles/StylerX-Styler_for_OBS_Studio.git
cd StylerX-Styler_for_OBS_Studio

# 2. Download Qt6 dev package for Windows from obs-deps
#    Extract to: StylerX/deps/qt6/

# 3. Configure
cd StylerX
mkdir build -Force; cd build
cmake .. -DCMAKE_PREFIX_PATH="../deps/qt6" -Dlibobs_DIR="../deps/obs-sdk/cmake"

# 4. Build
cmake --build . --config Release
```

> **Note:** The OBS SDK headers and CMake config are included in the repo under `StylerX/deps/obs-sdk/`. The Qt6 package is **not** tracked вҖ” download it separately from [obs-deps releases](https://github.com/obsproject/obs-deps/releases) (`windows-deps-qt6-*-x64.zip`).

## рҹ“Ұ Install

### Manual
```powershell
copy "StylerX\build\Release\styler-x.dll" "%ProgramFiles%\obs-studio\obs-plugins\64bit\"
copy "StylerX\build\Release\styler-x.pdb" "%ProgramFiles%\obs-studio\obs-plugins\64bit\"
```

### Via CLI
```powershell
python stylerx-cli\stylerx.py install --build-dir StylerX\build
```

### Verify Installation
```powershell
python stylerx-cli\stylerx.py doctor
```

## рҹҡҖ Usage

1. Launch **OBS Studio**.
2. Go to **Tools вҶ’ StylerX Studio**.
3. The dock panel appears with three tabs:

<div align="center">

| Tab | Description |
|-----|-------------|
| рҹҺЁ **Colors** | Pick and edit 20+ theme colors in real time |
| рҹ“қ **QSS** | Raw stylesheet editor with syntax highlighting |
| рҹ”Қ **Inspector** | Hover to highlight, click to inspect OBS widgets |

</div>

## рҹ—ӮпёҸ Theme Storage

Themes are JSON files stored at:

```
%APPDATA%/StylerX/themes/
```

Each theme includes metadata and a color map:

```json
{
  "format_version": 1,
  "name": "My Theme",
  "author": "you",
  "created": "2026-07-06T17:00:00Z",
  "modified": "2026-07-06T17:30:00Z",
  "colors": {
    "background": "#1e1e1e",
    "accent": "#0078d4",
    "text": "#cccccc"
  }
}
```
